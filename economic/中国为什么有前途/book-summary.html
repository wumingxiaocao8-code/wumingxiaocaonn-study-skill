二、理论基础：世界市场体系（M1-M3）

1. 四类国家划分

类型代表出口什么依靠什么生存
中央国家美国美元+无形资产（文化/品牌/专利）信心——别人对其实力的信任
准中央国家西欧、日本精致高附加值工业品+货币理性——百年工业积累+研发能力
外围工业国中国、印度、越南劳动密集型工业制成品勤劳——靠廉价劳动力
原料提供国中东、非洲、拉美能源和原材料运气——脚底下的自然资源

2. 体系六大特点

① 无远弗届

全球化覆盖全球，冷战后30亿人口+全球一半土地加入体系

② 非常不公平

中央比外围远为富有且稳定；顺周期/逆周期双标：外围被要求紧缩，中央自己宽松

③ 要素流动选择性

信息>资本>商品>技术>人力资本——最难流动的恰是最关键的

④ 自我强化

短期"劫贫济富"：资本和人才向中央集中→强者恒强

⑤ 周期性自我更新

长期"损有余而补不足"：创新→扩散→规模反转→中央转移

⑥ 结构鲁棒性

二战后仅几个经济体从外围挤入准中央，体系结构极其稳定

梵天神梦境：美帝国是做梦的主体，所有人都是梦境中的角色。外围国家的一切进步都在增强美元的购买力。脱离体系极难——外围精英以讲英语、移民美国为荣。

三、三外路线：中国崛起的引擎与代价（M4-M5）

五条政策组合

#政策具体内容
1汇率贬值人民币从2.5大幅贬至8.27，锚住美元11年，压低所有要素价格
2吸引外资超国民待遇（税收/土地/环境/基础设施优惠），外资主导出口加工
3鼓励出口出口退税、行政手段、加入WTO打开全球市场
4资本管制经常项目可兑换但资本项目不放开，拒绝投机性资本，欢迎产业资本
5强制结汇企业外汇必须卖给银行→银行头寸限制→央行做最后买家→4万亿外储

四利 vs 六弊

✅ 四利

  • 就业：3000万直接+2000万间接就业岗位
  • 工业化：加速技术溢出和产业能力积累
  • 资本积累：强制结汇克服发展中国家资本不足缺陷
  • 改革推动：WTO入世谈判倒逼国内体制改革

❌ 六弊

  • 挤压本土工业：民企被歧视，国企被锁死
  • 环境和资源代价：高耗能高污染
  • 过度依赖外部市场
  • 内部分配失衡：地区间/阶层间差距拉大
  • 纳贡4000亿/年：外资赚10%，外储收益接近0%
  • 不可持续：猛犸象压断树枝

🔗 齿轮咬合：三外路线 × 土地财政

三外路线 → 外汇占款(=基础货币大投放) → M2 高速增长(15-20%/年) → 银行大量可贷资金
三外路线释放流动性
银行可贷资金 → 开发商贷款+个人按揭 → 房价上涨 → 土地出让金(峰值8.7万亿)
房地产成为货币蓄水池
土地出让金 → 基建投资 → 城市面貌改善 → 地价再涨
土地财政自我强化
关键交叉补贴 → 住宅用地高价拍卖 → 补贴工业用地低价 → 维持出口竞争力
三外路线 ⇄ 土地财政：两个齿轮互相咬合

三外路线的四个国内根源

  1. 工业化使命——找到出口市场作为工业化的第一推动力
  2. 体制根源——全能主义遗产（社会渗透力）+ 分税制下地方官员"企业家化" + 中美体制互补（"天生一对"）
  3. 思潮因素——新自由主义经济学在1990s中国学术界的影响
  4. 要素禀赋——人口规模 = 廉价劳动力 + 规模经济 + 内需潜力
中美"天生一对"的成立与解体
成立条件：中国弱小 + 美国强大
解体条件：中国不那么弱小 + 美国不那么强大
→ 2008金融危机：暴露依附风险；2018贸易战：暴露政治上限

四、产业升级四种模式（M7-M8）

维度🚗 汽车🚄 高铁🌬️ 风电💻 芯片
买方 个人消费者 政府（铁道部） 发电企业（政府可影响） 被封锁（不让买）
外方态度 愿意合资，锁死技术 愿意卖但不情愿转让技术 愿意卖，被国产化率逼着教 根本不让卖（禁令）
中方策略 合资要求（形同虚设） 战略买家（唯一买方压价） 国产化率要求(1204公告) 被迫自主替代
结果 ❌ 30年没转移技术 ✅ 10年世界第一 ⚠️ 零件学会了，部分成功 🔄 加速中，部分突破
核心教训 国企无创新激励+买方是个人不可控 政府当战略买家+利用外资竞争 国产化率门槛逼出供应链溢出 外部封锁倒逼自研+举国体制
国产化率成功四条件
① 技术可拆解（核心技术在零件层面而非系统集成）
② 政府控买方（能强制要求采购国产）
③ 产业未成熟（上升期，技术壁垒未封死）
④ 外方有竞争（你不教总有别人教）

国企激励困境

国企高管的真实选择
A. "做对"：冒风险搞研发 → 成功收益归继任者，失败责任归自己 → 预期收益 负
B. "坐稳"：跟外资合资收分红 → 稳稳完成短期考核 → 预期收益 正
结果是 B 压倒 A → 产业升级缺乏市场主体

五、人民币汇率与国际化（M9）

一句话总结：对内重新定价商品与人的价格，对外从卖全球转向买全球——本质是从"世界打工者"变成"世界股东"。

资本项目开放四大辩论 vs 2026现实

焦点正方反方2026现实
① 利弊权衡 开放有利，大国不怕 风险巨大，历史上小国教训 各对一半：方向对但节奏被风险拖住了
② 改革顺序 同步推进 先内后外 接近反方：内改先行，外放渐进
③ 以开放促改革 WTO成功可复制 金融领域行不通 反方赢：对"拆墙"有效，对"盖楼"无效
④ 时间表 需要才有推进动力 保留政策灵活性 反方赢：无时间表，方向明确节奏模糊

债市外资占比~1.7%~23%

大机构不投中国的四因

  1. 规则不稳定——2021教培一夜覆灭的创伤还没愈合
  2. 资本管制（"阀门"）——进来容易，出去不一定容易
  3. 公司治理与透明度——财务造假成本低，退市力度不足
  4. 回报率不及预期——过去10年沪深300年化~3-5% vs 标普500 ~12%

七、资本输出与国际投资规则（M11-M12）

邓宁五阶段论

中国2014年起对外投资>吸收外资 → 进入第四阶段（净资本输出国），从外围工业国走向准中央国家。

关键概念速查

缩写全称一句话解释
BITBilateral Investment Treaty
双边投资协定		中国与某国签的投资保护"合同"，规定双方投资者在对方国家的权利

投资者-国家争端解决		BIT中最关键的条款——外国企业可以直接起诉东道国政府（不用通过本国政府外交途径）
MFNMost-Favored Nation

CAIComprehensive Agreement on Investment

ICSID国际投资争端解决中心ISDS最常用的仲裁庭，位于华盛顿，属世界银行
作者"差别化互惠"策略 vs 2026三重现实路径

但宏大构想未实现：WIO(世界投资组织)未成立、中式BIT范本未推出、"国家高于市场"理念未被接受

八、能源安全与原材料定价权（M13）

🔑 全书最激进观点：高油价有利于中国崛起

四本账：中国应托住油价在60-80美元/桶
战略账 → 高油价 → 俄/伊/委有钱反美 → 分散美国精力 → 中国赢得战略空间
经济账 → 多付370亿买油 → 产油国多赚1200亿 → 买中国工业品 → 出口收回成本
资源账 → 低买高储 → 增强定价权 + 金融杠杆 + 防止国际市场"逼空"
环境账 → 高油价 → 加速新能源技术突破 → 30年后中国不依赖进口能源

2022俄乌战争验证了"战略账"（美国被消耗），但也暴露局限（美国借机团结欧洲）。

铁矿石 vs 稀土：为何一个失败一个成功？

维度铁矿石（五连败）稀土（成功控价）
中国角色买家（进口方）卖家（出口方）
全球供给结构三巨头垄断70%市场中国一度占全球产量95%
国内结构100多家钢企各自为战，大钢企转手套利，中小钢企不配合200多家企业互相杀价→政府限产限出口→整合

九、对外援助（M14）

安哥拉模式（成功案例）

中国贷款 → 安哥拉以石油资源担保
中国工程队修路/盖房/建港口 → 安哥拉基础设施大幅改善
安哥拉经济高速增长 → 用石油还债
中国获石油份额+工程利润 | 安哥拉获基建+发展 | 民众获住房+就业

五个改进方案

  1. 重视人的作用——从"见物不见人"到教育/医疗/培训，派大学生"洋下乡"
  2. 改革决策体制——从"九龙治水"到统一协调机构+制定对外援助法
  3. 集中树典型——从"撒胡椒面"到集中100块钱给1-2个国家→品牌效应
  4. 援外+投资结合——自我造血，金融危机救援（别人做不了的→最有利可图）
  5. 改进宣传——让国内民众理解"美元不能用在国内"

十、结论：战略潜能与风险（M15）

六大战略潜能

① 产业链编辑能力

用自身市场规模塑造周边国家的不对称依赖，改善地缘政治处境

② 外汇储备杠杆化

从买美债到控股全球优质跨国公司，从债主变股东

③ 人民币国际化

修正国际货币格局，瓦解美元霸权根基

④ 大宗商品定价权

托油价、控稀土、整合铁矿石谈判

⑤ 广告捆绑式外宣

中国企业海外广告附带中国文化输出，用市场资源获取媒体影响力

⑥ 援外+资本输出

在体系外围扮演金融危机救援者

打通经脉：从"九龙治水"到统一指挥

改革前：归口管理制度割裂
经济线 （商务部/央行/发改委/财政部） ↛ 外事线 （外交部/安全部/总参）
2013年后：习近平兼任多个领导小组组长
打破制度壁垒 → 一带一路/亚投行/金砖银行得以出台
