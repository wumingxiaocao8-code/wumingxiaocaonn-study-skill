## 核心概念链
世界市场体系（中央-外围）→ "三外路线"（外贸/外资/外汇）→ 人民币国际化 → 资本输出 → 能源与原材料安全 → 对外援助 → 战略潜能与风险

### 详细概念网络
- **体系理论**：套叠型中央-外围结构 → 四类国家 → 喷泉池资金循环
- **三外路线**：引资 → 出口 → 结汇 → 4万亿外储 → 买美债（纳贡4000亿/年）
- **土地财政链接**：三外路线释放流动性 → 房地产蓄水池 → 工业用地交叉补贴
- **产业升级**：高铁模式（战略买家） vs 汽车模式（合资失败） vs 风电模式（国产化率）
- **人民币国际化**：从卖全球到买全球，从打工者到股东
- **外汇储备**：美元空头悖论 → 逼空风险 → 变债权为股权
- **全球投资规则**：BIT × ISDS × MFN → 差别化互惠策略
- **能源/原材料**：高油价战略（用370亿撬动1200亿财富转移） vs 铁矿石五连败 vs 稀土控价
- **援外**：安哥拉模式 → 援外+投资+贸易三位一体
