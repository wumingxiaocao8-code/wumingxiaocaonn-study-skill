# 《中国为什么有前途》概念图

## 全书核心逻辑链
```
世界市场体系(中央-外围) → 中国的位置(外围工业国→准中央)
          ↓
三外路线(外贸+外资+外汇) → 北冥神功(积累4万亿外储)
          ↓
代价：纳贡4000亿/年 + 土地财政锁死 + 国强民弱
          ↓
六脉神剑(经脉不通) → 各章"招数"打通经脉
          ↓
产业链编辑 → 人民币国际化 → 资本输出 → 定价权 → 援外品牌
          ↓
打通制度壁垒(归口管理) = 从九龙治水到统一指挥
```

## 核心概念详细关联

### 世界市场体系
- **四类国家**：中央(美国/信心) → 准中央(欧日/理性) → 外围工业(中国/勤劳) → 原料国(运气)
- **喷泉池循环**：外围赚美元→买美债→资金回流美国→美国消费→外围继续出口
- **六大特点**：无远弗届 / 不公平(顺/逆周期双标) / 要素流动选择性 / 自我强化 / 周期性自我更新 / 结构鲁棒性
- **顺周期vs逆周期**：外围被要求紧缩，中央自己宽松

### 三外路线
- **五条政策**：贬值+引资+鼓励出口+资本管制+强制结汇
- **四利**：就业/工业化/资本积累/改革推动/和平
- **六弊**：挤压本土/环境代价/过度依赖/分配失衡/纳贡4000亿/不可持续
- **齿轮咬合**：三外路线→流动性→房地产蓄水池→土地财政→补贴工业用地→维持出口竞争力

### 产业升级四种模式
- **汽车(失败)**：买方为个人消费者→合资30年核心技术没转移→国企无创新动力
- **高铁(成功)**：政府为唯一买方→战略买家策略逼技术转让→10年世界第一
- **风电(部分成功)**：政府设国产化率门槛(1204公告)→供应链溢出→5年外资份额从85%→3%
- **芯片(进行中)**：被封锁→只能自研→华为昇腾+中芯7nm，但差距仍在

### 人民币汇率与国际化
- **2005汇改**：从8.26缓慢升值→创造无风险套利空间→游资涌入
- **四大辩论**：利弊权衡 / 改革顺序 / 以开放促改革 / 时间表
- **先挂钩后脱钩**：学美元当年跟黄金挂钩→再跟石油挂钩

### 外汇储备
- **三位一体制度**：强制结汇+银行头寸限制+钉住美元→4万亿外储
- **美元空头悖论**：买80%美元→配成50%美元→实际上是结构性做空美元
- **逼空风险**：强美元+资本外逃+非美资产贬值→自我强化→但2022年未发生（资本管制兜底）

### 资本输出与投资规则
- **邓宁五阶段论**：中国进入第四阶段(净对外投资为正)
- **ISDS**：企业告政府的权利→中山富诚诉尼日利亚(2021)首胜
- **CAI**：中欧投资协定→文本完成→2021年被冻结→2026年欧盟无意重启
- **差别化互惠**：跟发达国家低标准(留政策空间) vs 跟发展中国家高标准(保投资安全)

### 能源与原材料
- **高油价战略**：托油价60-80美元→俄伊委有钱反美→牵制美国→中国赢战略空间
- **铁矿石五连败**：国内一盘散沙(大钢企转手套利→小钢企不服)→被三巨头各个击破
- **稀土控价**：95%全球产量→限产限出口→价格从3万/吨涨到几十万→WTO败诉但政策不变

### 对外援助
- **安哥拉模式**：以资源为担保→贷款+基建→受援国还债→品牌效应
- **集中树典型**：从"撒胡椒面"到100块钱给1-2个国家→"列入名单的都发了"
- **金融危机救援**：别人做不了的→最有利可图→救其急比救其穷更重要
