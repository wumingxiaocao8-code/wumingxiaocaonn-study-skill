# 《货币权力与人》概念图

## 核心概念链
民本主义 → 人与物 → 政府与市场 → 中心与外围 → 可贸易/非贸易
        ↘ 货币国际化 → 美元霸权 → 人民币国际化
                      ↘ 美元体系 → 利率 → 债务
                                  ↘ 金融权势 → 全球化
                                  ↘ 汇率

## 概念详细关联
### 民本主义
- 前导概念：天命论、民为邦本、天视民视
- 相关概念：人本主义（个体本位）、民主（过程导向）
- 核心创新：四组关系（人物/中心外围/政市/可贸非贸）
- 关键区分：民本 = 整体本位 + 结果导向 vs 人本 = 个体本位 + 权利导向

### 人与物
- 前导概念：劳动价值论（马克思）→ 作者补充消费端
- 核心命题：财富来源=人的劳动+人的消费，不是物的堆积
- 政策应用：苏联重生产轻消费→失败；中国生产消费并重→成功
- 延展：人口聚集→规模效应→更易致富（反马尔萨斯）

### 市场作为公共产品
- 核心创新：市场不是自然存在，而是政府提供的公共基础设施
- 反驳：古典自由主义（政府扭曲市场）和传统马克思主义（政府取代市场）
- 公式：政府能力 × 政府效率 = 国家财富
- 实证：政府规模与人类发展指数正相关（图1-3）
- 应用于：解释为什么弱国无强市场

### 中心与外围
- 传统结构主义：中心剥削外围（不平等交换）
- 民本主义修正：剥削 + 共生 = 双重关系
- 三个维度的人才掠夺：精英流失 + 市场挤压 + 资源诅咒
- 反直觉发现：中心国家债务率高反而安全（本币债）；外围债务率低反而危险（外币债）
- 融资模式对比：FDI vs 国际债务 vs 本国储蓄

### 可贸易与非贸易
- 核心区分：可贸易=财富创造，非贸易=财富分配
- 一价定理适用范围：仅限可贸易品
- PPP批判：没区分可贸/非贸，概念有害
- 政策应用：发展可贸易部门是脱贫核心
- 动态视角：技术变迁使非贸易品可贸易化→全球贫富分化加剧

### 货币本质
- 三层论：抽象单位→记账体系→可转让表征
- 区别古典货币商品说：货币不是物，是可转让的债权
- 政策含义：法币时代，货币价值仅取决于央行政策+公众信心

### 美元霸权
- 幂律分布：54/25%规律，网络效应自我强化
- 四大好处：铸币税乘数、危机转嫁、SWIFT霸权、体系压制挑战者
- 代价：美元=美国最大可贸易品→挤压制造业和人才培养
- 与第一章关联：美元特权=美国的"资源诅咒"

### 德日货币国际化对比
- 德国成功：央行独立+隐形冠军+政治自主+欧洲一体化
- 日本失败：央行依附+二元经济+政治不独立+次序错误
- 核心教训：先改革后开放，先区域化后国际化
- 六大金融改革：利率市场化、债市、打破垄断、公司治理、监管、汇率弹性

### 后布雷顿森林体系缺陷
- 三大缺陷：侵蚀美国制造业/外围外源性风险/扭曲微观激励
- 补救方案：改革SDR（锚定商品）+ 主权资产抵押机制
- 黄金价格=全球对美帝国信心的反向指标

### 央行与货币政策
- 央行独立性的四阶段：私有→国有→新自由主义独立→后危机搁置
- 核心批判：独立是幌子，中心和外围双重标准
- 非常规货币政策（QE）→ 只涨股价不涨物价 → 贫富分化 → 民粹

### 远端利率与人口
- 三大利率理论批判：作者以"消费"替代"投资"作为资金需求端
- 人口春秋比（0-24 vs 45-64）：正相关于远端利率
- 负利率根源：少子老龄化→总需求不足

### 金融权势与全球化
- 华尔街俘获华盛顿：献金+旋转门
- 去监管三部曲：利率自由化→混业→衍生品去监管
- 季度资本主义：波音案例
- 新自由主义破产：言行不一
- 大象曲线：中产阶级萎缩→民粹崛起
- 中国角色：不是搭便车，是支付了昂贵车票

### 汇率
- 巴萨效应批判：剔除非贸易品，只看可贸易品比较
- 决定汇率三大因素：出口结构（制造vs资源）+ 国家能力 + 人口年龄结构
- 核心论断：货币汇率是人的能力与信用的表现
- 广义美元区概念：美元指数取决于整个美元区，不只是美国
- 三外路线代价：金融抑制+土地财政转嫁成本

### 债务
- 中心vs外围：债务率高≠风险大，关键是**币种**
- 本币债安全，外币债危险
- 中国债务特点：内债+对应巨大资产→总体风险可控
- 评级霸权：发行人付费→利益冲突→"公器私用"

### 全书方法论
- 三种"错误的求商法"：GDP/人口（马尔萨斯）、债务/GDP（去杠杆焦虑）、投资/储蓄（利率理论）
- 共同错误：用孤立、静态的指标分析动态、复杂的社会系统
