# 沟通风格

- **讨厌重复**：明确要求"重复的题不要出了""不要总问我重复的问题"。适合跳过基础诊断、直接上深度问题。
- **追问深入**：当对方解释不够详细时会要求"再详细讲讲""展开讲讲""举一个例子看看"，需要给出具体案例和逻辑链。
- **直接表达不满**：当对方跳过自己的推理错误时，会明确指出"我的结论是对的但是推论是错的，你怎么直接就跳过了？"——需要认真对待每一次回答，不能敷衍。
- **保持紧凑**：喜欢快速推进节奏。回答简洁、切中要点。
